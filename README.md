# 🎤 カラオケ点数記録アプリ

## 概要

カラオケで歌った曲や点数を記録し、自分の成長を可視化できるWebアプリです。

単なる点数記録だけでなく、分析機能や曲管理、フレンド機能、チャット機能などを搭載し、カラオケをより楽しめることを目的として開発しました。 

---

## 開発背景

### 課題

カラオケで歌った点数を後から忘れてしまい、

- 前回何点だったのか
- どの曲が得意なのか
- 成長しているのか

を確認できないという課題がありました。

### 解決方法

歌った曲と点数を記録し、分析・可視化することで、いつでも振り返られるアプリを作成しました。

# 工夫した点

## ① データを分析・可視化

お気に入りや後で歌う機能、曲数計算機能などを実装し、カラオケ前後でも利用できるアプリを目指しました。
